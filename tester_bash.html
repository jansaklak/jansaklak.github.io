Code to test C++ programs.

Manual:

./tester [options]

Options:

-o FOLDER Set output folder (OUTPUT default)
-i FILE Set input file (./a.out default)
-v,fromFile FILE Set file with many input files written
--args FILE Set file with args to run
--cmp FILE Set file to compare output
--omp Use OpenMP
--mpi Use MPI
-minCore NUM Set min core NUM (Parallel only)
-c,-maxCore NUM Set min core NUM (Parallel only, default 2)
-s,-silent Silent mode
-h, -help Open help
