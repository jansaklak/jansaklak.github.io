MovieDB

Web application designed to provide users with information about movies. It is built using Node.js with the Express.js framework, EJS templating engine, and utilizes a SQLite database for data storage.

Features.

  • Movie Search: Users can search for movies by title.
  • Movie Details: Display detailed information about selected movies, including title, release date, and overview.

Technologies Used.

  • Backend: Users can search for movies by title.
  • Frontend: Display detailed information about selected movies, including title, release date, and overview.
  • Database: SQLite for storing movie data.

Project Structure.

The project follows a standard MVC (Model-View-Controller) architecture:

  • Controllers: Handle the application's logic and interact with the database.
  • Models: Define the data structures and interact with the database.
  • Views: EJS templates that render the user interface.
  • Routes: Define the application's endpoints and map them to controller functions.
  • Public: Contains static assets like CSS files.

Setup and Installation

  1. Clone the Repository:

    git clone https://github.com/jansaklak/MovieDB.git
    cd MovieDB

  2. Install Dependencies.

    npm install

  3. Start the Application

    npm start

The application will run on http://localhost:3000
