Overview

This project implements a genetic algorithm to optimize the stacking of various 3D shapes—such as cuboids, cones, spheres, and cylinders—within Blender. The primary objectives are to achieve a stable structure with a specified target height and to visualize the evolution of these structures over successive generations.

Prerequisites:

  • Python
  • Blender

The project comprises the following components:

  • Shape Classes: Definitions of various 3D shapes with methods to calculate volume, base surface areas, and center of mass.
  • Algorithm: Implementation of the genetic algorithm to evolve populations of shape stacks towards optimal configurations.
  • Function: A function to visualize the stacked shapes within Blender.

Each shape class inherits from the base Blocks class and implements methods to compute:

  • Volume (vol): Calculates the volume of the shape.
  • Upper Base Surface Area (upper_base_surf): Computes the surface area of the upper base.
  • Lower Base Surface Area (lower_base_surf): Computes the surface area of the lower base.
  • Center of Mass (get_center_of_mass): Determines the center of mass based on the current height in the stack.

The available shapes are:

  • Cuboid
  • Cone
  • Sphere
  • Cylinder
The GeneticAlgorithm class manages the evolution process with the following parameters:
  • pop_size - Population size.
  • generations - Number of generations to evolve.
  • mutation_rate - Probability of mutation.
  • target_height - Desired height of the stacked structure.
Key methods include:
  • create_random_block(): Generates a random stack of shapes.
  • create_random_pop(): Creates an initial population of random shape stacks.
  • fitness_function(stack): Evaluates the fitness of a shape stack based on contact surface area, total volume, center of mass height, and deviation from the target height.
  • selection(pop): Selects the top-performing individuals from the population.
  • crossover(par1, par2): Performs crossover between two parent stacks to produce offspring.
  • mutation(figure): Applies mutation to a shape stack.
The render_blocks(blocks) function utilizes Blender's bpy module to visualize the stacked shapes. It:
  • Clears the current Blender scene
  • Iterates through the list of shapes, adding them to the scene at calculated positions.
  • Adjusts the position and orientation of each shape based on its properties and the stacking order
  1. Initialize the Genetic Algorithm: Create an instance of the GeneticAlgorithm class with desired parameters.

    algorytm = GeneticAlgorithm(pop_size=100, generations=30, mutation_rate=0.1, target_height=10)

  2. Run the Algorithm: Append the run() method to Blender's frame change handler to execute the algorithm.

    bpy.app.handlers.frame_change_pre.append(algorytm.run())

  3. Set Animation Frames: Define the number of frames for the animation.

    frames = 50
    bpy.context.scene.frame_end = frames

  4. Start the Animation: Play the animation within Blender to observe the evolution of the shape stacks.
