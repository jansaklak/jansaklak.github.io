Code to simulate work of logic gates written in Python.

Instrukcja użycia:

./tester [opcje]

Opcje:

-o FOLDER Ustaw folder wyjściowy na FOLDER (domyślnie OUTPUT)
-i FILE Ustaw plik wejściowy na FILE (domyślnie ./a.out)
-v,fromFile FILE Ustaw plik z listą programow do uruchomienia
--args FILE Plik z argumentami dla testowanych programów
--cmp FILE Ustaw plik oczekiwany wyjściowy z którym ma być zgodne wyjście
--omp Skorzystaj z OpenMP
--mpi Skorzystaj z MPI
-minCore NUM Ustaw minimalną liczbę rdzeni na NUM
-c,-maxCore NUM Ustaw maksymalną liczbę rdzeni na NUM (domyślnie 2)
-s,-silent Tryb cichy - brak wypisywania na ekran
-h, -help Wyświetl tę instrukcję
